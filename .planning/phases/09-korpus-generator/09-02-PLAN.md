After creating all 5 panels and setting their AssembledTransform, build the preview list:
```csharp
// Build 3D preview -- transform each flat panel to assembled position, then extrude by thickness
_previewBreps = new List<Brep>();
foreach (var panel in panels)
{
    Brep assembled = panel.FlatBrep.DuplicateBrep();
    assembled.Transform(panel.AssembledTransform);

    // Determine extrusion direction based on panel orientation:
    // Boden/Deckel: extrude in +Z (thickness goes up)
    // LinkeSeite/RechteSeite: extrude in +X direction (toward center)
    // Rueckwand: extrude in -Y direction (toward front)
    // All panels are thin boxes: use Surface.CreateExtrusion on each face

    // Simpler approach: create a box Brep directly from assembled corner points.
    // Since each flat panel after transform is a planar surface at the correct position,
    // we can extrude it by the panel's thickness in its normal direction.

    // Get the face normal of the assembled planar Brep
    Vector3d normal = assembled.Faces[0].NormalAt(
        assembled.Faces[0].Domain(0).Mid,
        assembled.Faces[0].Domain(1).Mid);

    // Create extrusion path along the normal by thickness
    Point3d extStart = Point3d.Origin;
    Point3d extEnd = extStart + normal * panel.Thickness;
    LineCurve extPath = new LineCurve(new Line(extStart, extEnd));

    Brep solid = assembled.Faces[0].CreateExtrusion(extPath, true);
    if (solid != null)
    {
        _previewBreps.Add(solid);
    }
}
```

IMPORTANT: The face normal direction matters for correct extrusion:
- **Boden** at Z=0: flat face normal is (0,0,1) or (0,0,-1). If normal points down, the extrusion goes downward. For correct visual, the Boden panel should extrude upward (+Z) by MS. After PlaneToPlane transform from WorldXY, the normal should be +Z. Verify the extrusion direction gives a visually correct box.
- **Deckel** at Z=H-MS: should extrude upward by MS to reach Z=H.
- **LinkeSeite** at X=0: should extrude inward (+X) by MS.
- **RechteSeite** at X=B-MS: should extrude outward (+X) by MS to reach X=B. Actually RechteSeite origin is at B-MS, so extruding along the face normal (+X or -X) by MS. If normal is +X, it extrudes to B. If -X, it goes wrong direction. The PlaneToPlane transform determines this.

If the face normal direction causes incorrect extrusion for some panels, use a sign convention: always extrude along the outward-facing normal. Alternatively, check if the resulting solid bounding box looks correct and flip if needed. Simplest robust approach: after extrusion, just accept whatever direction the normal gives -- for visual preview purposes both directions create a valid solid box.
